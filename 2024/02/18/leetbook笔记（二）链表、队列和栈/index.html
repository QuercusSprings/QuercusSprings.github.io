删除操作

  • 删除中间结点

    1. 找到cur的上一个结点prev及其下一个结点next
    2. 连接prev到cur的下一个结点next

  • 删除第一结点

    1. 直接将下一个结点分配为head结点

  • 删除最后结点

    1. 置倒数第二个结点的后继指针为空

队列

  • 队列是先入先出（FIFO）的数据结构，插入（insert）操作也称作入队（enqueue），新元素始终被添加在队列的末尾。 删除（delete）操作也被称为出队（dequeue)。 只能移除第一个元素。

  • ```c++
    #include
    //队列应支持两种操作，入队和出队，需要一个索引指出起点
    class MyQueue {
    private:
    // 数据域
    vector data;
    // 起点
    int p_start;
    public:
    MyQueue() {p_start = 0;}
    //先入先出，入队时起点不变
    bool enQueue(int x) {
    data.push_back(x);
    return true;
    }
    //出队时，先入先出，即起点后移
    bool deQueue() {
    if (isEmpty()) {
    return false;
    }
    p_start++;
    return true;
    };
    //得到起点
    int Front() {
    return data[p_start];
    };
    //检查队列是否空
    bool isEmpty() {
    return p_start >= data.size();
    }
    };

    int main() {
    MyQueue q;
    q.enQueue(5);
    q.enQueue(3);
    if (!q.isEmpty()) {
    cout << q.Front() << endl;
    }
    q.deQueue();
    if (!q.isEmpty()) {
    cout << q.Front() << endl;
    }
    q.deQueue();
    if (!q.isEmpty()) {
    cout << q.Front() << endl;
    }
    }
    //随着出队即起始指针的移动，浪费了越来越多的空间。 当我们有空间限制时，这将是难以接受的。这就需要循环队列。

栈

  • 栈是后入先出（LIFO）的数据结构

  • //可用可变数组实现堆栈结构 could use variable-array 
class MyStack{
    private:
    vector<int> data; //store elements
    public:
    // 常用函数 判空 判顶 压栈 出栈
    bool isEmpty(){
        return data.empty();
    }
    int top(){
        return data.back();
    }
    void push(int x){
        data.push_back(x);
    }
    bool pop(){
        if(isEmpty()){
            return false;
        }
        data.pop_back();
        return true;
    }
};
